NY State COVID-19 Pipeline

See Project on Github

Description:
Emails an automated COVID-19 update that includes a line chart displaying Albany County's "new postivie" cases over time and a 5 day summary table.

Project created with:

  • NY State COVID-19 Testing API
  • Python 3.8.5
  • pandas 1.1.0
  • matplotlib 3.3.1
  • apache-airflow 1.10.11
  • Postgres 2.3.5
